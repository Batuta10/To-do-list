### Installation

[Node.js](https://nodejs.org/) v9+ to run.

Install the dependencies and devDependencies and check and start the server.

```sh
$ node -v
exit v9.x
$ nodemon -v
exit 1.x
$ npm i in the front and back directory
exit ...
```
